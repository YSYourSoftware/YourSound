# YourSound

As another entry of the [YourSoftware series](https://gloriousglider8.github.io/your), YourSound is an open source sound engine, with a very permissive license.

## Project Structure

- `core` - Core Engine, can easily be integrated into other projects.
- `factory` - Factory content.
- `standalone` - Standalone application.
- `web` - Web integration with WASM.
- `plugin` - DAW Plugin support using JUCE.
