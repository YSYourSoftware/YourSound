# YourSound

As another entry of the [YourSoftware series](https://gloriousglider8.github.io/your), YourSound is an open source sound engine, with a very permissive licence.

## Project Parts

- `core` - Core Engine, can easily be integrated into other projects.
- `factory` - Factory content.
- `standalone` - Standalone application.
- `plugin` - DAW Plugin support using JUCE.
- `libmgr` - Library manager frontend.
- `libtest` - Testing of player loading and interactions.
